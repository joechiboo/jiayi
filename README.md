# Jiayi's Games

歡迎來到我的遊戲世界！🎮

## 遊戲列表

### 2248 連線遊戲
在 5×7 的棋盤上連接相同數字或相鄰倍數，挑戰你的策略思維！支援斜角連線，享受數字消除的樂趣。

[🎮 開始遊戲](https://joechiboo.github.io/2248/)

## 技術特色
- 響應式設計，支援各種裝置
- 純 HTML5/CSS3/JavaScript 實現
- 美觀的漸變背景和動畫效果
- 直觀的遊戲操作體驗
